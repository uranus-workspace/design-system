# Consume Uranus Design System

Use the published packages from the Uranus monorepo:

- `@uranus-workspace/design-system` — primitives (Button, Input, Card, …).
- `@uranus-workspace/blocks` — composite layout blocks built on primitives.
- `@uranus-workspace/tokens` / `@uranus-workspace/tailwind-config` — brand tokens and Tailwind v4 preset.

Import components from the package exports; use semantic CSS variables (for example `--color-primary`), not raw ramp steps. See the docs site `/docs` for usage, a11y notes, and Storybook-aligned examples.
