title: NotificationList

`NotificationList` é a lista que vive dentro do `Popover` do sino no `AppHeader`. Cada item tem título, descrição opcional, timestamp e flag `unread`. Quando `unread: true`, mostra um ponto azul + `aria-current="true"`.


<ComponentPreview name="notification-list-default" align="start" />

## Uso

```tsx
import { NotificationList } from '@uranus-workspace/blocks';
import { Popover, PopoverContent, PopoverTrigger, Button } from '@uranus-workspace/design-system';
import { Bell } from 'lucide-react';

<Popover>
  <PopoverTrigger asChild>
    <Button variant="ghost" size="icon" aria-label="Notificações">
      <Bell aria-hidden className="size-4" />
    </Button>
  </PopoverTrigger>
  <PopoverContent className="p-0" align="end">
    <NotificationList
      items={notifications}
      onMarkAllRead={markAllRead}
    />
  </PopoverContent>
</Popover>
```

## Props

- `items` — lista. Cada item pode ter `onSelect` para virar um botão clicável.
- `onMarkAllRead` — só aparece quando há ao menos um item `unread`.
- `emptyState` — sobrescreve o `EmptyState` padrão ("You're all caught up").
