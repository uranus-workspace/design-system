`SignUpForm` usa **`signUpFormSchema`** (Zod) e **`react-hook-form`** sobre os mesmos primitivos `Form` da lib base. Aceite dos termos é validado pelo schema.

## Uso

```tsx
import { SignUpForm } from '@uranus-workspace/blocks';
import type { SignUpFormValues } from '@uranus-workspace/blocks';

<SignUpForm
  signInHref="/sign-in"
  termsLabel={
    <>
      Aceito os <a href="/terms">Termos</a> e a <a href="/privacy">Privacidade</a>.
    </>
  }
  onSubmit={async (values: SignUpFormValues) => {
    await registerAction(values);
  }}
/>
```

```ts
import { signUpFormSchema } from '@uranus-workspace/blocks';

signUpFormSchema.safeParse(body);
```

## Produtos só OAuth

`credentials="hidden"` oculta nome, email, senha e envio correspondente — use com `OAuthProviderButton` / `socialProviders`:

```tsx
import { OAuthProviderButton, SignUpForm } from '@uranus-workspace/blocks';

<SignUpForm
  credentials="hidden"
  signInHref="/sign-in"
  socialProviders={<OAuthProviderButton provider="google">…</OAuthProviderButton>}
/>
```

## Valores

`SignUpFormValues` é exportado; corresponde a `z.infer<typeof signUpFormSchema>`.

## Medidor de força

A pontuação 0–4 é apenas visual (UI), enquanto o mínimo de 8 caracteres vem do **schema Zod**. Para políticas enterprise, valide também no servidor com o mesmo `signUpFormSchema` ou um schema estendido.
