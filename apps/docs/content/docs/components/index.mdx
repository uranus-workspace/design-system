Os componentes vivem em `@uranus-workspace/design-system` e têm origem em [shadcn/ui](https://ui.shadcn.com/docs/components). Nós somos donos do código — `pnpm dlx shadcn@latest add <nome>` escreve novos primitivos direto em `packages/design-system/src/components/`, onde customizamos para casar com os tokens e a marca Uranus.

Todo componente chega com:

1. Uma implementação TypeScript estrita
2. Testes `jest-axe`
3. Stories do Storybook com o addon a11y em `error`
4. Uma página de documentação MDX (esta seção)
