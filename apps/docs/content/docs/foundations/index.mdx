title: Foundations
description: The invariants — color, typography, spacing, motion, accessibility, and voice — that every Uranus interface shares.
---

Foundations are the decisions every Uranus interface inherits. They live in `@uranus-workspace/tokens` and flow outward through `@uranus-workspace/tailwind-config` into every component and every page.

> **Placeholder values.** Until the Uranus branding manual PDF is ingested, foundations ship with neutral defaults. When the manual lands, we update `packages/tokens/src/*.ts` and regenerate the CSS variables — no component code changes.
