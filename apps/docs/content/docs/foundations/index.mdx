title: Foundations
description: Os invariantes — cor, tipografia, espaçamento, movimento, acessibilidade e voz — que toda interface Uranus compartilha.
---

Foundations são as decisões que toda interface Uranus herda. Elas moram em `@uranus-workspace/tokens` e fluem para fora através do `@uranus-workspace/tailwind-config` para cada componente e cada página.

Cada token tem origem no **Manual de Marca Uranus 2026** — a fonte visual única da verdade mantida pelo time de marca. Mudar um valor de foundation é uma decisão de marca: edite `packages/tokens/src/*.ts`, regenere o CSS, e todo consumidor pega automaticamente.

- [Cor](/docs/foundations/color) — paleta cósmica + highlight lilás + gradientes cósmicos
- [Tipografia](/docs/foundations/typography) — Lexend Exa / Poppins
- [Espaçamento](/docs/foundations/spacing)
- [Movimento](/docs/foundations/motion)
- [Acessibilidade](/docs/foundations/accessibility)
