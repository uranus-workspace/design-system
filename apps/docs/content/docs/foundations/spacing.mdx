---
title: Espaçamento
description: A escala de espaçamento baseada em 4px usada para padding, margin e gap.
---

O espaçamento usa uma base de 4px (`0.25rem`). Os valores são escritos em `rem` para que escalem com o tamanho de fonte preferido pelo usuário.

Use utilitários do Tailwind — `p-2`, `gap-4`, `space-y-6` — em vez de valores pixelados arbitrários. Se um valor não existe na escala, pergunte-se se o design realmente precisa de um novo token antes de recorrer a `p-[17px]`.
