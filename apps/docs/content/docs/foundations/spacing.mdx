---
title: Spacing
description: The 4px-based spacing scale used for padding, margin, and gap.
---

Spacing uses a 4px base (`0.25rem`). Values are authored in `rem` so they scale with the user's preferred font size.

Use Tailwind utilities — `p-2`, `gap-4`, `space-y-6` — rather than arbitrary pixel values. If a value doesn't exist in the scale, ask whether the design actually needs a new token before falling back to `p-[17px]`.
