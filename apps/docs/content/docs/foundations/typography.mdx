---
title: Typography
description: Type scale, font families, and weight tokens for Uranus interfaces.
---

Type is rendered with the system font stack until the Uranus branding manual defines a display typeface. The scale ranges from `xs` to `6xl`, expressed in `rem` so it adapts to the user's root font size.

## Scale

| Token | Size | Line height |
|---|---|---|
| xs | 0.75rem | 1rem |
| sm | 0.875rem | 1.25rem |
| base | 1rem | 1.5rem |
| lg | 1.125rem | 1.75rem |
| xl | 1.25rem | 1.75rem |
| 2xl | 1.5rem | 2rem |
| 3xl | 1.875rem | 2.25rem |
| 4xl | 2.25rem | 2.5rem |
| 5xl | 3rem | 1.1 |
| 6xl | 3.75rem | 1.05 |

Weights available: `400`, `500`, `600`, `700`.
