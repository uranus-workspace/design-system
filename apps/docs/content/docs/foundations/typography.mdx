O manual de marca Uranus 2026 pareia duas famílias:

- **Lexend Exa** — primária / display. Escolhida pela amplitude horizontal ("imensidão do horizonte espacial"). Usada em headings, momentos de marca e exibições numéricas.
- **Poppins** — secundária / corpo. Uma sans-serif geométrica que equilibra sofisticação e utilidade em UI densa.

Ambas são carregadas via Google Fonts. Em apps Next.js usamos `next/font/google` para inline; no Storybook elas vêm de um link de CDN no `preview-head.html`.

## Famílias

| Papel | Token | Stack |
| --- | --- | --- |
| Display | `font-display` | `Lexend Exa, Poppins, system-ui` |
| Corpo | `font-sans` | `Poppins, system-ui` |
| Mono | `font-mono` | `ui-monospace, SF Mono, Menlo…` |

```tsx
<h1 className="font-display text-5xl tracking-tight">Uranus</h1>
<p className="font-sans text-base text-muted-foreground">
  Um hub de serviços flexível e escalável.
</p>
```

## Pesos

Lexend Exa: `300`, `400`, `500`, `600`, `700`, `800`.
Poppins: `300`, `400`, `500`, `600`, `700`.

Use as classes do Tailwind `font-light`, `font-normal`, `font-medium`, `font-semibold`, `font-bold`, `font-extrabold`.

## Espaçamento entre letras

Headings display usam por padrão `-0.02em` (apertado) para um tom disciplinado e técnico. Em momentos de marca onde Lexend Exa é protagonista, permita `-0.04em` (mais apertado) para respeitar os contraformas largos.

## Faça / Não faça

- **Faça** o par entre headings Lexend Exa e corpo Poppins em conteúdo longo.
- **Faça** uso de `font-display` em exibições numéricas (dashboards, KPIs) para reforçar a marca.
- **Não** misture Lexend Exa com uma terceira tipografia display. O par é intencional.
- **Não** use Lexend Exa em caixa alta em tamanhos pequenos — a legibilidade sofre abaixo de 14px.
