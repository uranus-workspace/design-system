O **Uranus Design System** é a fonte única da verdade para como os produtos da [Uranus Technologies](https://uranus.com.br/) parecem, funcionam e se comportam. Toda superfície — o site institucional em [uranus.com.br](https://uranus.com.br/), o produto OmniFisco, ferramentas internas — consome dos mesmos tokens, componentes e linguagem de movimento documentados aqui.

> _"Mais do que estética, a nova Uranus se posiciona como um hub de serviços flexível e escalável."_ — Manual de Marca Uranus, 2026

## A marca em um olhar

A identidade Uranus é construída sobre uma **paleta cósmica** e um par tipográfico modernista, codificados no manual de marca 2026:

| Token | HEX | Papel |
| --- | --- | --- |
| Azul Profundo | `#000328` | Superfície escura primária (`brand-950`) |
| Azul Marinho | `#082d71` | Marca / wordmark (`brand-800`) |
| Azul Turquesa | `#5dddfa` | Acento, foco, destaques (`brand-400`) |
| Lilás Claro | `#f8ddfc` | Highlight decorativo (`accent-200`) |
| Preto Clássico | `#000000` | Neutro base (`neutral-950`) |
| Branco Puro | `#ffffff` | Neutro base (`neutral-50`) |

A tipografia é **Poppins** — um sistema de família única, do corpo do texto até os headings de marca. Fundos combinam Preto Clássico com os três azuis através dos utilitários `bg-cosmic`, `bg-aurora`, `bg-nebula` e `bg-galaxy`.

## O que há aqui dentro

- **Foundations** — primitivos visuais e comportamentais (cor, tipografia, espaçamento, movimento, acessibilidade, voz) compartilhados por toda interface.
- **Componentes** — primitivos de UI focados e reutilizáveis (`@uranus-workspace/design-system`) construídos sobre shadcn/ui, Tailwind CSS v4 e Motion. O catálogo completo do shadcn/ui (54+ componentes) já vem incluído.
- **Blocos** — layouts compostos de nível mais alto (`@uranus-workspace/blocks`) montados a partir dos primitivos.
- **Auth** — pacote OIDC (`@uranus-workspace/auth`) com presets Keycloak, screens pré-conectados e middleware Next.js opt-in.
- **AI** — superfícies para aplicações de IA (`@uranus-workspace/ai`) — chat, composer, streaming, raciocínio e planejamento.
- **Padrões** — guia opinativo para fluxos recorrentes como formulários, empty states e navegação.
